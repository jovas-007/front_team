# Data Quality Dashboard

## Estructura
- `backend_team/` — Backend (Django 4.2 + DRF).
- `front_team/` — Frontend (Node 18.19).
- `venv/` — Entorno virtual Python 3.12.1.

## Versiones
- Python 3.12.1
- Node 18.19
- Django 4.2.22
- DRF 3.16.1
- pandas 2.2.2, numpy 2.0.2

## Cómo iniciar
1. `python -m venv venv` y activar.
2. `pip install -r backend_team/requirements.txt`.
3. (En próximos pasos) crear proyecto Django y scaffold del front.
